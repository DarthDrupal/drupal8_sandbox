Module to play with the new Drupal8 APIs.

- Routing API: Basic routing example (Hello World Page)
- Routing API: Routing example getting arguments from url
- Config API: Read values from yml file
- Config API: Create default configuration using hook_install
- Config API: Edit values in yml file using ConfigFormBase
- Form API: Simple form example using FormBase
- Form API: Simple redirection example on form submission
